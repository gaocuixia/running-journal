# 个人跑步记录网站

一个记录跑步感悟的个人网站，支持文章的添加、修改、删除和分类展示功能。

## 功能特性

- ✍️ 写感悟：发布新的跑步感悟文章
- 📝 编辑文章：修改已发布的文章
- 🗑️ 删除文章：删除不需要的文章
- 🏷️ 分类展示：按训练日志、比赛感悟、装备评测、跑步心得分类查看
- 🔍 排序功能：按发布日期排序（最新/最早）
- 💾 本地存储：所有数据保存在浏览器本地存储中

## 如何访问本地网站

### 方法一：使用 ngrok 临时暴露（推荐）

ngrok 是一个简单易用的工具，可以将本地服务临时暴露到公网，适合临时分享给朋友访问。

#### 安装 ngrok

1. 访问 [ngrok 官网](https://ngrok.com/download) 下载适合 macOS 的版本
2. 解压下载的文件，将 ngrok 可执行文件移动到 `/usr/local/bin` 目录（或其他已添加到 PATH 的目录）

#### 使用步骤

1. 确保你的本地服务器已经在运行（通常是 `python3 -m http.server 8000`）
2. 打开终端，运行以下命令：
   ```bash
   ngrok http 8000
   ```
3. 执行成功后，会看到类似以下输出：
   ```
   Forwarding                    http://abc123.ngrok.io -> http://localhost:8000
   Forwarding                    https://abc123.ngrok.io -> http://localhost:8000
   ```
4. 将生成的公网 URL（如 `http://abc123.ngrok.io`）分享给朋友即可访问

**注意**：ngrok 免费版的 URL 会在每次启动时变化，且有连接时间限制，适合临时使用。

### 方法二：路由器端口映射（长期访问）

如果你有公网 IP，可以通过路由器的端口映射功能，让朋友长期访问你的本地网站。

#### 步骤

1. 确保你的本地服务器运行在固定端口（如 8000）
2. 找到你的路由器管理地址（通常是 `192.168.1.1` 或 `192.168.0.1`）
3. 登录路由器管理界面，找到「端口映射」或「虚拟服务器」设置
4. 添加映射规则：
   - 外部端口：80（或其他可用端口）
   - 内部 IP 地址：你的电脑在局域网中的 IP（如 `192.168.1.100`）
   - 内部端口：8000（本地服务器运行的端口）
   - 协议：TCP
5. 保存设置后，朋友可以通过你的公网 IP 访问网站

**注意**：
- 你需要知道自己的公网 IP（可以通过 `curl ifconfig.me` 查看）
- 部分 ISP 可能不提供固定公网 IP，需要使用 DDNS 服务
- 安全性：开放端口会增加被攻击的风险，建议仅在需要时启用

### 方法三：云服务部署（稳定长期访问）

将网站部署到云服务器，提供稳定的公网访问。

#### 推荐平台

1. **GitHub Pages**：免费托管静态网站，适合个人博客
   - 将网站文件上传到 GitHub 仓库
   - 开启 GitHub Pages 功能
   - 访问 `https://yourusername.github.io/repo-name/`

2. **Vercel** / **Netlify**：免费托管静态网站，支持自动部署
   - 连接 GitHub 仓库
   - 自动构建和部署
   - 生成公网 URL

3. **Cloudflare Pages**：免费托管，支持自定义域名

#### 部署步骤（以 GitHub Pages 为例）

1. 在 GitHub 上创建新仓库
2. 将本地网站文件推送到仓库
3. 进入仓库设置，找到「Pages」选项
4. 选择分支（通常是 `main`）和目录（通常是 `/`）
5. 点击「Save」，等待部署完成
6. 访问生成的 URL（如 `https://yourusername.github.io/running-journal/`）

## 本地开发

### 启动本地服务器

```bash
python3 -m http.server 8000
```

然后在浏览器中访问 `http://localhost:8000`

### 文件结构

```
├── index.html          # 网站主页面
├── style.css           # 样式文件
├── script.js           # JavaScript 功能文件
└── README.md           # 说明文档
```

## 浏览器兼容性

- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+

## 数据存储

所有文章数据保存在浏览器的 `localStorage` 中，不会上传到服务器。
